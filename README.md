# 🚀 LeetCode Tracker

Track your friends' LeetCode progress in real-time with streaks, rankings, and problem statistics.

## 🔗 Live Demo
(Open index.html in browser)

## ✨ Features

- 📊 Track multiple users' LeetCode stats
- 🔥 Daily streak calculation
- 🧠 Easy / Medium / Hard breakdown
- 🏆 Ranking display
- 🔄 Update user stats dynamically
- ❌ Remove users
- ⚡ Fast and responsive UI

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (Vanilla JS)
- LeetCode API (3rd-party)

## ⚙️ How It Works

- Takes a LeetCode username as input
- Fetches user data from a public API
- Processes submission calendar to calculate streak
- Dynamically renders user cards
- Sorts users based on total problems solved
